Flixshare Backend API Documentation

GET /users

Retrieve a list of users from the database.

Example usage:

GET /users

POST /api/signup

Create a new user account.

Request Body:

{
  "email": "user@example.com",
  "username": "user123",
  "password": "password123"
}

POST /api/login

Authenticate and log in a user.

Request Body:

{
  "username": "user123",
  "password": "password123"
}

POST /logout

Log out the current user.
